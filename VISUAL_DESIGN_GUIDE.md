## Responsive Breakpoints

- **Mobile (< 640px)**
  - 1 column pricing
  - Full-width inputs
  - Visible hamburger menu

- **Tablet (640px - 1024px)**
  - 2-3 columns (depends on section)
  - Adjusted spacing
  - Hidden desktop-only elements

- **Desktop (> 1024px)**
  - 3-column pricing grid
  - Popular card: lg:scale-105
  - Full feature set visible
  - Spacing optimized
